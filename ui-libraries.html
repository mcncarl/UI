1. 桌面端（PC）组件库

企业级组件库

Ant Design

由蚂蚁金服开发的企业级UI设计语言和React组件库

适用场景：
  • 中后台管理系统
  • 企业级应用
  • 数据密集型应用
  • 需要统一设计规范的大型项目
使用方法：

通过npm或yarn安装：

npm install antd

在React项目中引入组件：

import { Button } from 'antd';
官方网站 →

Element UI

由饿了么前端团队开发的基于Vue的桌面端组件库

适用场景：
  • 基于Vue开发的中后台系统
  • 需要美观统一UI的企业应用
  • 追求开发效率的项目
使用方法：

通过npm或yarn安装：

npm install element-ui

在Vue项目中引入：

import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';

Vue.use(ElementUI);
官方网站 →

Material UI

基于Google的Material Design设计规范的React组件库

适用场景：
  • 想要遵循Material Design的应用
  • React应用程序
  • 响应式Web应用
使用方法：

通过npm或yarn安装：

npm install @mui/material @emotion/react @emotion/styled

在React项目中使用：

import Button from '@mui/material/Button';

function App() {
  return ;
}

轻量级组件库

Bulma

基于Flexbox的现代CSS框架，无JavaScript依赖

适用场景：
  • 需要简洁但漂亮UI的项目
  • 希望避免JavaScript依赖的项目
  • 快速开发原型
  • 小型到中型网站
使用方法：

通过CDN引入：

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css">

或通过npm安装：

npm install bulma
官方网站 →

Tailwind UI

基于Tailwind CSS的官方高级UI组件集合

适用场景：
  • 使用Tailwind CSS的项目
  • 需要灵活定制样式的应用
  • 追求原子化CSS开发体验
  • 从头构建UI系统的项目
使用方法：

先安装Tailwind CSS：

npm install tailwindcss

配置Tailwind CSS，然后将Tailwind UI组件复制到您的项目中：

<div class="bg-white shadow overflow-hidden sm:rounded-lg">
  <!-- 复制Tailwind UI组件的HTML -->
</div>

2. 移动端组件库

Vant

有赞前端团队开发的轻量、可靠的移动端Vue组件库

适用场景：
  • 移动端Vue项目
  • 电商类应用
  • 需要高性能UI组件的项目
  • 需要兼顾功能和视觉效果的应用
使用方法：

通过npm安装：

npm i vant

在Vue项目中引入：

import { Button } from 'vant';
import 'vant/lib/index.css';

// 注册组件
app.use(Button);
官方网站 →

Mint UI

饿了么团队开发的基于Vue的移动端组件库

适用场景：
  • 基于Vue的移动端应用
  • 需要扁平化风格UI的项目
  • 需要轻量级解决方案的应用

通过npm安装：

npm install mint-ui

在Vue项目中引入：

import Mint from 'mint-ui';
import 'mint-ui/lib/style.css';

Vue.use(Mint);
官方网站 →

Cube UI

滴滴团队开发的精致移动端Vue组件库

适用场景：
  • Vue移动端应用
  • 需要丰富交互效果的项目
  • 追求高质量细节表现的应用
  • 支持按需引入的项目
使用方法：

通过npm安装：

npm install cube-ui

配置Vue项目以使用Cube UI：

// vue.config.js
module.exports = {
  css: {
    loaderOptions: {
      stylus: {
        'resolve url': true
      }
    }
  },
  pluginOptions: {
    'cube-ui': {
      postCompile: true,
      theme: true
    }
  }
}
官方网站 →

3. 小程序组件库

WeUI WXSS

微信官方为小程序设计的UI库，与微信原生风格一致

适用场景：
  • 微信小程序开发
  • 需要原生微信体验的应用
  • 追求简洁高效的小程序
使用方法：

下载WeUI WXSS源码到本地项目：

git clone https://github.com/Tencent/weui-wxss.git

在小程序中引入样式：

/* 在app.wxss中 */
@import 'path/to/weui-wxss/dist/style/weui.wxss';
官方仓库 →

Vant Weapp

有赞团队开发的轻量级、高性能微信小程序UI组件库

适用场景：
  • 微信小程序开发
  • 电商类小程序
  • 需要丰富组件的小程序
  • 注重性能的项目
使用方法：

通过npm安装：

npm i @vant/weapp -S --production

在小程序中使用组件：

// 在app.json或页面json中配置
"usingComponents": {
  "van-button": "@vant/weapp/button/index"
}
官方网站 →

Taro UI

基于Taro框架开发的多端UI组件库，支持微信、支付宝、百度等小程序

适用场景：
  • 使用Taro开发的多端应用
  • 需要同时适配多个小程序平台
  • React开发风格的小程序项目
  • 需要统一多端体验的应用
使用方法：

首先安装Taro框架，然后安装Taro UI：

npm install taro-ui

配置项目以使用Taro UI：

// 引入组件样式
import 'taro-ui/dist/style/index.scss'

// 使用组件
import { AtButton } from 'taro-ui'

render() {
  return 按钮文案
}

4. 跨平台组件库

NutUI

京东风格的跨端组件库，同时支持Vue和小程序开发

适用场景：
  • 需要同时开发移动端Web和小程序
  • 京东系产品或类似电商应用
  • Vue技术栈项目
  • 需要高性能和丰富功能的应用
使用方法：

通过npm安装：

npm i @nutui/nutui

在Vue项目中引入：

import NutUI from "@nutui/nutui";
import "@nutui/nutui/dist/style.css";

app.use(NutUI);
官方网站 →

uni-app UI

基于uni-app的跨平台UI组件库，支持H5、App和小程序

适用场景：
  • 使用uni-app框架的项目
  • 需要同时开发多端应用
  • 追求一套代码多端使用的项目
  • 需要丰富组件的跨平台应用
使用方法：

在uni-app项目中安装：

npm install @dcloudio/uni-ui

在页面中使用组件：

// pages.json
{
  "easycom": {
    "autoscan": true,
    "custom": {
      "^uni-(.*)": "@dcloudio/uni-ui/lib/uni-$1/uni-$1.vue"
    }
  }
}

// 页面中使用
